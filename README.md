# Kapselki

Kapselki to lekka gra iOS o podworkowych wyscigach kapsli. Gracz wybiera postac, nawierzchnie i prowadzi kapsel po kredowej trasie, pstrykajac go z wyczuciem mocy, spinu i kontroli.

Strona aplikacji: https://lpociask.github.io/kapselki/

## Co jest w projekcie

- `Kapselki/` - kod aplikacji SwiftUI i SceneKit.
- `Kapselki.xcodeproj/` - projekt Xcode.
- `preview/` - zrzuty ekranu aplikacji.
- `concept-art/` - materialy koncepcyjne postaci.
- `docs/` - statyczna strona publikowana przez GitHub Pages.

## Start lokalnie

1. Otworz `Kapselki.xcodeproj` w Xcode.
2. Wybierz docelowy symulator i uruchom aplikacje.
3. Strone aplikacji mozna podejrzec lokalnie, otwierajac `docs/index.html` w przegladarce.

## Status

To poczatek repozytorium: aplikacja ma juz pierwszy ekran, wybor kapsla, samouczek, widok trasy i materialy promocyjne na GitHub Pages.
